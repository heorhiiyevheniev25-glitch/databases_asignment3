# databases_asignment3
У базі створено функцію calculate_order_total(p_order_id int), яка обчислює загальну суму замовлення на основі таблиці order_items. Сума рахується як quantity * price. Якщо замовлення не має товарів, функція повертає 0. Процедура create_order(p_customer_id int) створює нове замовлення для існуючого клієнта. Перед вставкою нового рядка в таблицю orders процедура перевіряє, чи є клієнт із customer_id. Процедура add_product_to_order(p_order_id int, p_product_id int, p_quantity int) додає товар до замовлення. Вона бере актуальну ціну товару з таблиці products, перевіряє, що кількість більша за нуль, перевіряє наявність достатньої кількості товару на складі, додає запис у order_items і зменшує products.stock_quantity. Реалізовано два тригери. Перший тригер автоматично оновлює orders.total_amount після змін у таблиці order_items. Він працює після INSERT, UPDATE і DELETE. Другий тригер записує інформацію про створення нового замовлення в таблицю order_log.
